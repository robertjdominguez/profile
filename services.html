Student Centric
Education Gamification

Student Centric is a gamification platform for students that encourages studying and repetition via a gamification model for the classroom. Teachers can create online quizzes where the students earn XP for completing each quiz.

Cheese Wrangler
Dynamic Trip Manifests

The "Cheese Wrangler" system is a dynamically integrated bus manifest system that can be totally controlled via SMS. Using Twilio's API, leaders can create trips and students can easily register for the trip itself. Admins have a dashboard that allows them to monitor the progress of trips and see every detail of the trip's manifest.
